# アヴァロン ONLINE

準備なしで遊べるオンライン・アヴァロン。合言葉を共有するだけで同じ部屋に入って遊べます。
Node + Express + Socket.IO + 素のフロント、DBなし、自作エンジン。

## 遊び方
1. 1人が名前を入れて「部屋を作る」→ 4文字の合言葉が出る
2. 仲間はその合言葉と名前を入れて「部屋に入る」（5〜10人）
3. ホストが「ゲーム開始」→ 役職が自動配布
4. 各自「確認した」を押すとクエスト開始
5. リーダーがメンバー選択 → 全員で賛成/反対 → 選抜者が成功/失敗
6. 善が3勝で暗殺フェーズ、悪が3勝 or 5連続否決で悪の勝利

## ルール設定（ノーマル・シンプル版）
- 善：マーリン / パーシヴァル / アーサーの忠臣
- 悪：モルガナ / 暗殺者 / モードレッドの手先
- マーリンには悪が全員見える
- パーシヴァルにはマーリンとモルガナが（区別なしで）見える
- 7人以上は第4クエストのみ2フェイルで失敗（盤面に ★ 表示）

## ローカル起動
```
npm install
npm start
```
http://localhost:3000

## Render（無料）デプロイ
1. このフォルダを GitHub リポジトリに push
2. Render で New → Web Service → リポジトリを選択
3. Build Command: `npm install`
4. Start Command: `npm start`
5. Instance Type: Free
6. 環境変数 PORT は Render が自動で渡すので設定不要

無料プランは無操作でスリープしますが、プレイ中は起き続けます。
（スリープ復帰後は最初のアクセスに数十秒かかることがあります。状態はメモリ保持なので、サーバー再起動でゲームはリセットされます）
